## 1. רקע כללי

חברת Kids & Fun הפקות בע"מ עוסקת בניהול פעילויות לילדים בשלושה תחומים עיקריים:

| תחום | אחוז מהפעילות | תיאור |
|------|---------------|-------|
| צהרונים | 60% | ניהול מערך צהרונים שנתי |
| קייטנות | 30% | ניהול קייטנות בחופשות |
| אירועים | 10% | ימי הולדת, גיבוש, אירועי בתי ספר |

**מטרת המערכת:** בניית פלטפורמה אחודה לניהול כלל הפעילויות, הכוללת ניהול לקוחות, ילדים, עובדים, נוכחות, תשלומים ותקשורת.

## 2. מודולי המערכת

### 2.1 מודול ניהול (Admin)
- [ ] ניהול משתמשי מערכת והרשאות
- [ ] ניהול סניפים
- [ ] הגדרות כלליות

### 2.2 מודול לקוחות וילדים
- [ ] ניהול אנשי קשר (הורים/לקוחות עסקיים)
- [ ] ניהול ילדים
- [ ] קישור ילדים למשפחות
- [ ] מידע רפואי ואלרגיות
- [ ] אנשי קשר לחירום

### 2.3 מודול צהרונים
- [ ] הגדרת מסגרות צהרון (שנתי)
- [ ] ניהול קבוצות וגילאים
- [ ] רישום ילדים לצהרון
- [ ] מעקב נוכחות יומית
- [ ] חיובים חודשיים

### 2.4 מודול קייטנות
- [ ] הגדרת קייטנות (קיץ, פסח, חנוכה)
- [ ] ניהול מחזורים/שבועות
- [ ] ניהול קבוצות
- [ ] רישום ילדים למחזורים
- [ ] תיעוד פעילויות יומיות

### 2.5 מודול אירועים
- [ ] סוגי אירועים (יום הולדת, גיבוש, בי"ס)
- [ ] חבילות ומחירונים
- [ ] ניהול אירוע (תאריך, מיקום, לקוח)
- [ ] שיבוץ צוות לאירוע
- [ ] ניהול ציוד

### 2.6 מודול עובדים
- [ ] ניהול פרטי עובדים/מדריכים
- [ ] שיוך לסניף/קבוצה
- [ ] זמינות ומשמרות

### 2.7 מודול כספים
- [ ] יצירת חיובים (חודשי/חד פעמי)
- [ ] רישום תשלומים
- [ ] מעקב חובות
- [ ] דוחות כספיים

### 2.8 מודול תקשורת
- [ ] שליחת SMS/WhatsApp
- [ ] תבניות הודעות
- [ ] היסטוריית הודעות

### 2.9 מודול דוחות
- [ ] דוח נוכחות
- [ ] דוח כספי
- [ ] דוח רישומים
- [ ] דוחות מותאמים

## 3. ישויות המערכת (Entities)

### 3.1 ישויות ליבה

| ישות | שם בעברית | תיאור | הערות לקוח |
|------|-----------|-------|------------|
| User | משתמש | משתמשי המערכת + הרשאות | |
| Contact | איש קשר | הורה / לקוח עסקי | |
| Child | ילד | פרטי ילד | |
| Employee | עובד | מדריך / עובד | |
| Branch | סניף | סניף פעילות | |
| Document | מסמך | קבצים ומסמכים מצורפים | |

### 3.2 ישויות צהרונים

| ישות | שם בעברית | תיאור | הערות לקוח |
|------|-----------|-------|------------|
| Afternoon | צהרון | מסגרת צהרון (שנתית) | |
| AfternoonGroup | קבוצת צהרון | קבוצה במסגרת הצהרון | |
| AfternoonChild | רישום לצהרון | קישור ילד לצהרון | |
| Attendance | נוכחות | רישום נוכחות יומי | |
| MonthlyCharge | חיוב חודשי | חיוב למשפחה | |

### 3.3 ישויות קייטנות

| ישות | שם בעברית | תיאור | הערות לקוח |
|------|-----------|-------|------------|
| Camp | קייטנה | הגדרת קייטנה | |
| CampCycle | מחזור | שבוע/מחזור בקייטנה | |
| CampGroup | קבוצת קייטנה | קבוצה במחזור | |
| CampRegistration | רישום לקייטנה | רישום ילד למחזור | |
| CampActivity | פעילות | פעילות יומית בקייטנה | |

### 3.4 ישויות אירועים

| ישות | שם בעברית | תיאור | הערות לקוח |
|------|-----------|-------|------------|
| Event | אירוע | אירוע בודד | |
| EventType | סוג אירוע | קטגוריית אירוע | |
| EventPackage | חבילת אירוע | חבילה + מחיר | |
| EventStaff | צוות אירוע | שיבוץ עובדים לאירוע | |
| EventEquipment | ציוד אירוע | ציוד נדרש לאירוע | |

### 3.5 ישויות משותפות

| ישות | שם בעברית | תיאור | הערות לקוח |
|------|-----------|-------|------------|
| Payment | תשלום | רישום תשלום | |
| Message | הודעה | הודעה ללקוח | |
| Note | הערה | הערה על ילד/לקוח/אירוע | |
| MedicalInfo | מידע רפואי | אלרגיות, תרופות, מגבלות | |
| EmergencyContact | איש קשר לחירום | פרטי חירום של ילד | |
